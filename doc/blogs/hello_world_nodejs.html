Run hello.js in windows

How?

  1. Download the node.js installer in node.js officia web site and install it.
  2. In windows CMD, run node --version command line for check the node install correctly, you should get some return message like this v7.9.0, if not, copy the error message and google will help you.
  3. OK, if you install node.js correctly, now create a file with .js subffix such as hello.js in where you like.
  4. Let us suppose, there is hello.js in your computer with path D:\hello.js. Now, open you CMD ,type some command line into folder where hello.js is.
  5. Eventually, type command line node hello.js to run you javascript script.

Why?

  1. when you want to run some javascript script you should use node.js to execute it.Node.js is a Runtime Enviroment for javascript. For more simply, you could consider node.js is a program which could run javascript.
  2. Node.js provide a command line tool for run it. So, you could type node in CMD for run it. Go a step further, you could provide some argument for node.js for do something. Like node --version , you provide --version argument to told node.js you want to know which version of node.js, and the node.js will response you. And, node hello.js is told node.js you want to run a javascript file called hello.js, so node.js do it for you.
